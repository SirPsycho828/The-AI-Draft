# Design System — The AI Draft

> Generated by Design Overhaul skill. This document is the single source of truth for all design decisions.

## Design Direction

**Direction:** Draft Day — ESPN Draft Night meets Bloomberg Terminal
**Signature Element:** Continuous scrolling marquee ticker, rolling number counters, BREAKING pulse animations

## Animation

| Token | Value | Usage |
|-------|-------|-------|
| --duration-fast | 100ms | Micro-interactions (hover, focus, ticker) |
| --duration-normal | 200ms | State transitions (card expand, filter toggle) |
| --duration-slow | 350ms | Page transitions, reveals |
| --duration-ticker | 30s | Marquee ticker scroll speed (full loop) |
| --easing-default | cubic-bezier(0.4, 0, 0.2, 1) | General motion |
| --easing-spring | cubic-bezier(0.34, 1.56, 0.64, 1) | Bouncy card entrances |
| --easing-out | cubic-bezier(0, 0, 0.2, 1) | Exit animations |

**Signature Animations:**
1. **Marquee Ticker:** Continuous horizontal scroll at top of every page showing latest moves. 30s loop, seamless wrap, pauses on hover.
2. **Card Entrance:** Staggered slide-up + fade. Each card enters 50ms after the previous one. Spring easing for bounce.
3. **Number Roll:** Stats count from 0 to final value over 1.5s with ease-out. Triggered on scroll-into-view.
4. **Breaking Pulse:** Chartreuse border glow that pulses 3 times then settles. For new/breaking move events.
5. **Flash Border:** New feed items get a brief chartreuse border flash (200ms) when they enter the feed.
